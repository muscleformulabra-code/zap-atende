## Padrão geral de uma especialidade
descrição → valores → fecho → menu "Gostaria de marcar sua consulta?" →
- **Sim** → pede (Nome completo, Contato, Data nascimento, CPF) → atendente (handoff)
- **Não** → "Que pena, ficamos à disposição" → encerra (pausa automação)
